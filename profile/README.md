## 🧰 What we build

| Domain | Current focus |
| ------ | ------------- |
| **Server-Side Swift** | Micro-services with **Vapor** & **Hummingbird** (e.g. 🌨️ *SnowCover API*, powered by Sentinel data) · **SRM** — a Swift replacement for pm2 with auto-restart & startup helpers |
| **Local & Edge AI** | Jetson-ready fork of **llama.cpp** (CUDA 10.2) · **SwiftLlama** — idiomatic Swift wrapper for llama.cpp · **Neuronaut** — Swift wrapper for ByteDance's Trae Agent in Telegram |
| **Mobile & Cross-Platform** | iOS sports-tracking, mapping, navigation & AR prototypes · VR prototypes with **SwiftGodot** on Apple Vision / Meta Quest · React-Native + Python back-ends for LLM micro-services |
| **Game Tech** | **Godot 4 + SwiftGodot** R&D for 2D RPG/strategy games & terrain generation · *Unity as a Framework* integration to embed Unity into any iOS view · Home-IoT experiments incl. hardware selection & setup |
| **Home-Lab / DevOps** | Raspberry Pi / Jetson clusters for distributed LLM inference · NGINX, Docker, systemd & PM2 playbooks · automated deployment scripts and Telegram bot monitors |
